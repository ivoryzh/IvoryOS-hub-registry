# IvoryOS-hub-registry
Registry is the ingestion and validation layer for IvoryOS-compatible drivers.

## Using the Agent Evaluation Guide

See the [Agent Evaluation Guide](agent_evaluation_guide.md) for instructions on how to evaluate Python repositories for compatibility with IvoryOS.

To use the guide when evaluating a repository:
1. `git clone` the target repository you want to evaluate.
2. You can either copy the contents of the evaluation guide and paste it for your agent, or download the `agent_evaluation_guide.md` file and add it directly to the target repository. Either way works!
